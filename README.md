#ITR Accounting System

This project is a demonstration of an ITR (Income Tax Return) Accounting System. It showcases the use of SQL to design, build, and interact with a relational database. The project includes various SQL commands and operations to ensure the database is functional, efficient, and aligned with accounting system requirements.

Features:
1. Database schema creation for an accounting system

2. Implementation of core SQL commands (CREATE, INSERT, SELECT, UPDATE, DELETE)

3. Query optimization techniques

4. Use of constraints, relationships, and indexing to ensure data integrity

5. Sample data and queries to demonstrate real-world use cases

Purpose:
The primary goal of this project is to illustrate the practical application of SQL in building an accounting system, specifically focused on managing Income Tax Returns.
